Membrum

VST3 Synthesized Drum Machine

Version 0.1.0

Overview

Membrum is a synthesized drum machine that generates drum sounds from first principles using modal synthesis. Instead of playing back samples, it models the physical behavior of vibrating membranes, shells, and plates to create expressive, velocity-sensitive drum sounds that respond naturally to how you play.

Phase 1 delivers a single membrane drum voice with 5 parameters: Material, Size, Decay, Strike Position, and Level. The voice combines an impact exciter with a 16-mode modal resonator bank using circular membrane Bessel function ratios for physically accurate frequency relationships.

Features

  • Modal synthesis engine -- 16 parallel Gordon-Smith resonators modeling circular membrane vibration modes with Bessel function frequency ratios
  • Velocity-sensitive excitation -- Soft hits produce dark, muted tones; hard hits produce bright, punchy sounds with wider bandwidth excitation
  • Material control -- Sweep from woody (high damping, fast HF decay) to metallic (even decay, long sustain)
  • Size control -- Exponential frequency mapping from small (500 Hz) to large (50 Hz) with proportional mode scaling
  • Decay control -- Global decay time scaling that preserves material character
  • Strike position -- Center strike excites fundamental modes; edge strike excites higher-order modes for complex timbres
  • Real-time safe -- Zero allocations in the audio thread; all DSP pre-allocated
  • Cross-platform -- VST3 (Windows, macOS, Linux) + Audio Unit (macOS)

Installation

Windows

Copy the VST3 bundle to your VST3 folder:

C:\Program Files\Common Files\VST3\Krate Audio\Membrum\Membrum.vst3

macOS

Copy the VST3 and AU bundles:

/Library/Audio/Plug-Ins/VST3/Membrum.vst3
/Library/Audio/Plug-Ins/Components/Membrum.component

Linux

Extract and copy to your VST3 folder:

cp -r Membrum.vst3 ~/.vst3/
