Features

  • 10 unique delay modes including Tape, BBD, Digital, Shimmer, and more
  • 100 factory presets (10 per mode) for instant inspiration
  • Built-in preset browser with save, load, and search functionality
  • Tempo-synchronized delays with note value selection
  • Full parameter automation support
  • Cross-platform: VST3 (Windows, macOS, Linux) + Audio Unit (macOS)
  • Low-latency, real-time safe audio processing

Installation

Windows

Run the .exe installer. The plugin will be installed to:

C:\Program Files\Common Files\VST3\Krate Audio\Iterum\Iterum.vst3

macOS

Double-click the .pkg installer. Both formats will be installed:

/Library/Audio/Plug-Ins/VST3/Iterum.vst3
/Library/Audio/Plug-Ins/Components/Iterum.component

Linux

Extract the archive and copy to your VST3 folder:

cp -r Iterum.vst3 ~/.vst3/
